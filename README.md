
## API Reference

#### Create Data

```http
  POST /data
```

#### Update Data

```http
  PUT /data
```
#### Retrive all 

```http
  GET /data
```
#### Retrive one single entry

```http
  PUT /data/:id
```
| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |



#### Delete

```http
  DELETE /data/:id
```
